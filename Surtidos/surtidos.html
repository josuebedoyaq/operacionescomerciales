SURTIDOS

¿QUÉ ES EL SURTIDO?

Es el conjunto de artículos expuestos en el punto de venta susceptibles a ser adquiridos por los clientes. Se refiere a la variedad de artículos presentes, no a la cantidad de un producto determinado.

EL LINEAL

Medida longitudinal del espacio donde se exponen los productos para su venta en régimen de autoservicio. Este espacio está compuesto por todo el surtido a la venta: expositores, mostradores, vitrinas y cámaras frigoríficas de productos refrigerados y congelados.

NIVEL DEL SUELO:

Tiene de 70 a 80 cm de altura, el cliente para alcanzar el producto debe agacharse, por lo que este nivel no tiene muchas ventas (posibilidad de venta 13%).

NIVEL DE LAS MANOS:

Se encuentra a una altura de 80 a 120 cm, el esfuerzo para alcanzar el artículo es menor que el anterior ya que esta al nivel de las manos. Este tipo de nivel tiene mayor venta que el anterior (posibilidad de venta 26%).

NIVEL DE LOS OJOS:

Está a una altura de 120 a 170 cm del suelo, es el nivel más cómodo para el cliente ya que esta a la altura de los ojos. Este tipo de nivel tiene mucha mayor venta que los dos anteriores (posibilidad de venta 52%).
